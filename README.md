# 3-1 Converter (C# WinForms)

Простой и эффективный конвертер файлов. Сделал Оспанов Тимур Ерболович, 11-ВТИ-24.

## Основные функции:
* **Текст:** Конвертация .txt в .pdf.
* **Изображения:** Быстрое преобразование .png в .jpg.
* **Аудио:** Конвертация тяжелых .flac файлов в компактный .mp3.

## Технологии:
* **UI:** Windows Forms (.NET 8.0/6.0)
* **Библиотеки:** PdfSharp (PDF), NAudio & NAudio.Lame (Audio).

## Как запустить:
Перейдите в раздел **Releases** и скачайте последний `Converter.exe`. Установка не требуется.
